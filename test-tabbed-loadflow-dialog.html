Overview

This test page demonstrates the new tabbed Load Flow Dialog that supports both Pandapower and OpenDSS engines.

New Features:

  • Tabbed Interface: Switch between Pandapower and OpenDSS engines
  • Pandapower Tab: Full parameter set for Pandapower calculations
  • OpenDSS Tab: OpenDSS-specific parameters and algorithms
  • Unified Interface: Single dialog for both engines
  • Enhanced Backend: Improved OpenDSS parameter handling

// LoadFlowDialog.js - Now includes tabs for both engines // OpenDSSLoadFlowDialog.js - Dedicated OpenDSS dialog // loadflowOpenDss.js - Updated to work with tabbed interface // dialogInitializer.js - Enhanced initialization

Backend Changes:

# opendss_electrisim.py - Enhanced parameter handling # app.py - Updated to handle new OpenDSS parameters # New parameters: algorithm, max_iterations, tolerance, convergence, voltage_control, tap_control

New OpenDSS Parameters:

  • Algorithm: Admittance (Iterative) or PowerFlow (Direct)
  • Max Iterations: Maximum number of iterations
  • Tolerance: Convergence tolerance
  • Convergence: Normal or Accelerated
  • Voltage Control: Enable/disable voltage control
  • Tap Control: Enable/disable tap control

Usage Instructions

  1. Open the Load Flow Dialog from your Electrisim application
  2. Choose between Pandapower and OpenDSS tabs
  3. Configure parameters for your selected engine
  4. Click Calculate to run the simulation
  5. View results displayed on your diagram

Technical Notes

  • The dialog automatically detects which engine is selected
  • Parameters are validated before sending to backend
  • Results are displayed using the existing visualization system
  • Subscription checks are maintained for both engines
  • Fallback to legacy dialogs is supported for backward compatibility
