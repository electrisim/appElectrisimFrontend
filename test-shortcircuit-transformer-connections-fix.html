Short Circuit Transformer Connections Fix Test

Backend Errors Fixed

Original Backend Logs (Errors):

Trafo(s) [1, 6, 11, 16]: hv and lv connectors seem to be swapped Trafo3w 0: Nominal voltage on hv_side (400.0 kV) and voltage_level of hv_bus (bus 1 with voltage_level 30.0 kV) deviate more than +/- 30.0 percent. Trafo3w 0: Nominal voltage on mv_side (30.0 kV) and voltage_level of mv_bus (bus 0 with voltage_level 400.0 kV) deviate more than +/- 30.0 percent.

Root Cause Analysis:

  • Missing Functions: shortCircuit.js was calling updateTransformerBusConnections and updateThreeWindingTransformerConnections but these functions were not defined in its scope
  • Bus Connection Mismatch: Without the connection correction logic, transformers were sending incorrect bus assignments to the backend
  • Voltage Level Conflicts: Three-winding transformers had their high-voltage and medium-voltage buses swapped, causing validation errors in pandapower
  • Code Duplication Issue: The functions existed in loadFlow.js but were not shared with shortCircuit.js

Solution Applied:

✅ Added all missing helper functions to shortCircuit.js:

  • updateTransformerBusConnections - Sorts 2-winding transformer connections by voltage
  • updateThreeWindingTransformerConnections - Sorts 3-winding transformer connections by voltage (HV, MV, LV)
  • getConnectedBusId, getThreeWindingConnections, getTransformerConnections - Connection helper functions
  • getConnectedBuses, getImpedanceConnections - Line and impedance connection functions
  • parseCellStyle, validateBusConnections - Utility functions

Functions Added to shortCircuit.js

1. Regular Transformer Connection Fixing

The updateTransformerBusConnections function ensures that:

  • High voltage bus is correctly assigned to hv_bus
  • Low voltage bus is correctly assigned to lv_bus
  • Prevents "hv and lv connectors seem to be swapped" errors

2. Three-Winding Transformer Connection Fixing

The updateThreeWindingTransformerConnections function ensures that:

  • Highest voltage bus → hv_bus
  • Medium voltage bus → mv_bus
  • Lowest voltage bus → lv_bus
  • Prevents voltage level deviation errors in pandapower

3. Connection Detection Logic

Added helper functions for detecting element connections:

  • getConnectedBusId - Single bus connections
  • getThreeWindingConnections - Three-winding transformer edge connections
  • getTransformerConnections - Regular transformer edge connections
  • getConnectedBuses - Line connections (busFrom/busTo)
  • getImpedanceConnections - Impedance connections
